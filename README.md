# ProgramacionFuncional

El 12/12/2025 presentamos el trabajo de programación funcional sobre la Premier League 24-25. Uno de los aspectos que decidimos desarrollar se basa en el aprendizaje automático, por ejemplo, para predecir la diferencia de goles por partido.

On 12/12/2025 we presented the functional programming work on the 24-25 Premier League. One of the aspects that we decided to develop is based on machine learning, for example, to predict the goal difference per game.

Em 12/12/2025, apresentamos o trabalho de programação funcional da Premier League 24-25. Um dos aspectos que decidimos desenvolver é baseado em aprendizado de máquina, por exemplo, para prever a diferença de gols por jogo.
